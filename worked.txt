ALDST Project Work Completed So Far

Project Name:
ALDST - AI Learning Disability Support Tool

Project Goal:
The project is a web-based learning assistant for students with ADHD and dyslexia.
It helps users convert complex study material into simplified notes, flashcards,
translations, audio-assisted reading, and focus-based study sessions.

Technology Stack Used:
Frontend:
- React.js
- Tailwind CSS
- Vite
- Lucide React icons

Backend:
- FastAPI
- Python
- SQLAlchemy
- SQLite for local development
- PostgreSQL/Supabase-ready structure for deployment

AI and Processing:
- Local structured fallback for notes and flashcards
- Optional HuggingFace API support
- HuggingFace-ready translation models
- Browser speech synthesis for audio-assisted study

Deployment Direction:
- Frontend can be deployed on Vercel
- Backend can be deployed on Render or Railway
- Environment variables are supported for hosted backend and AI tokens

Deployment Notes:
For frontend deployment on Vercel, set:
VITE_API_URL=https://your-backend-url/api

For backend deployment on Render or Railway, set:
HF_API_TOKEN=your_huggingface_token

This token is required for live HuggingFace translation.
Without it, the app will still run but translation will show a configuration message.
